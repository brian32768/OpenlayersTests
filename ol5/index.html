OpenLayers 5 example

Inspired by the book "OpenLayers 3.x Cookbook Second Edition" but bearing in mind that OpenLayers 5 is refactored into many small files.

I am using the example from the OpenLayers 5 tutorial which uses the "import" statement. This threw me for a loop since import is not implemented in some browsers. I am now using Parcel, which precompiles the Javascript, processing the import properly.
